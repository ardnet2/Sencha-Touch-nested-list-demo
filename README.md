A demonstration of how the Nested List View in Sencha Touch.

Also, see the [Sencha Touch Nested List example][eg].

Create the Postgres database:

    createdb music_catalogue
    rake db:seed

## Deploying to Heroku ##

Run:

    git push heroku master
    heroku db:push
